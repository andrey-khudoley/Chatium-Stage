# Сайт ритуала "Наказание врагов"

## Описание проекта

Одностраничный лендинг для духовной практики - ритуала "Наказание врагов". Представляет собой интерактивный блокнот с кольцевым переплетом на скрепках, фотографией на скрепке и цветными закладками для навигации.

## Архитектура

### Структура файлов

```
/dev/site/130126/
├── index.tsx              # Главная точка входа, SSR-рендеринг
├── styles.tsx             # Переиспользуемые стили и head-теги
├── pages/
│   └── RitualPage.vue     # Основной компонент страницы
└── .CHATIUM-LLM.md       # Этот файл
```

### Технологии

- **Frontend**: Vue 3 (Composition API, `<script setup>`)
- **Styling**: TailwindCSS 3.4 (CDN)
- **Fonts**: 
  - Cormorant Garamond (заголовки, декоративный текст)
  - Raleway (основной текст)
- **Icons**: Font Awesome 6.7.2

## Дизайн-концепция

### Визуальная метафора

**Прелоадер**: Закрытая обложка блокнота бордового цвета с текстом и металлическими кольцами.

**Основной дизайн**: Открытый блокнот с:
- Металлическими кольцами слева
- Мистическим фото на скрепке вверху справа
- Цветными закладками сверху для навигации
- Кремовыми страницами с текстурой бумаги
- Эффектом открытия обложки с shift вправо

### Цветовая палитра

```
notebook-cover: #5A1F29    (обложка блокнота)
notebook-spine: #4A1721    (корешок обложки)
page-cream: #F8F4E8        (основной фон страниц)
page-shadow: #E8E0D0       (тени на страницах)
text-dark: #2C2416         (основной текст)
text-light: #5C4F3A        (вторичный текст)
gold-accent: #B8860B       (золотые акценты)
burgundy: #722F37          (важные элементы, цена)
ring-metal: #9A8B7A        (цвет металлических колец)
```

### Компоненты дизайна

1. **Прелоадер-обложка** - закрытая бордовая обложка с анимацией открытия
2. **Металлические кольца слева** - реалистичные с градиентами и внутренними тенями
3. **Фото на скрепке** - мистическое фото с ритуальной атмосферой
4. **Цветные закладки сверху** - 5 цветных вкладок (розовая, оранжевая, синяя, зеленая, желтая)
5. **Бумажная текстура** - кремовый фон с subtle паттерном
6. **Индикаторы страниц** - цветные точки внизу

## Структура контента

### Страницы (табы)

1. **Intro (О ритуале)** - Заголовок и введение
2. **Action (Действие)** - Что делает ритуал
3. **Who (Для кого)** - Целевая аудитория
4. **Result (Результат)** - Что получите
5. **Order (Заказать)** - Цена и призыв к действию

### Анимации

- **cover-open** - открытие обложки с shift вправо (1.2s)
- **content-reveal** - появление контента после открытия (0.8s, delay 1s)
- **photo-appear** - появление фото на скрепке (0.6s, delay 1.4s)
- **page-in/page-out** - переходы между страницами (0.5s)
- **float** - плавающая анимация для иконок (3s infinite)
- **tab transitions** - плавное изменение закладок при наведении

## Адаптивность

### Breakpoints

- **Mobile** (< 768px): Упрощенная навигация, уменьшенные отступы
- **Tablet/Desktop** (≥ 768px): Полная версия с крупными элементами

### Адаптивные элементы

- Размеры шрифтов (text-lg/text-xl → text-2xl/text-3xl)
- Отступы (px-4/py-8 → px-8/py-12)
- Размеры колец (w-8/h-8 → w-10/h-10)
- Ширина переплета (w-12 → w-16)

## Особенности реализации

### Vue композиция

- `isLoading` - флаг загрузки (управляет прелоадером)
- `activeTab` - текущая активная вкладка (intro/action/who/result/order)
- `ritualPhoto` - URL сгенерированного мистического фото
- `tabs` - массив закладок с цветами
- `onMounted()` - запуск таймера скрытия прелоадера (1.5s)

### Конфигурация Tailwind

Кастомная конфигурация с:
- Семейства шрифтов
- Цветовая палитра
- Кастомные анимации и keyframes
