# Модуль @app/ugc: UGC-файлы и права

Краткий справочник по модулю `@app/ugc` — работа с файлами кода UGC (исходники приложения) и правами доступа к ним. Типизация: `node_modules/@app/ugc/index.d.ts`. Пересекается по контексту с фидами, инбоксом и агентами (все работают в UGC-контексте).

## Назначение

**@app/ugc** — API для работы с файловой структурой UGC (код приложения в рантайме): поиск файла по пути или id, обновление содержимого (updateUgcFileSource, updateUgcFile, updateUgcFilesMulti), создание директорий, удаление файлов и директорий, а также управление правами доступа к UGC-файлам (listUgcFilePermissions, hasUgcFilePermission, createOrUpdateUgcFilePermission, deleteUgcFilePermission).

---

## Файлы UGC

- **findUgcFile(ctx, path)** / **findUgcFile(ctx, path, options)** — найти файл по пути.
- **findUgcFileById(ctx, id)** — найти файл по id.
- **updateUgcFileSource(ctx, path, source)** — обновить исходный код файла.
- **updateUgcFile** / **updateUgcFilesMulti** — обновление метаданных/нескольких файлов.
- **listUgcFiles(ctx, options)** — список файлов (путь или массив путей, либо UgcListUgcFilesOptions).
- **ensureUgcDirectory(ctx, path)** — создать директорию при необходимости.
- **deleteUgcFile(ctx, path)** — удалить файл.
- **deleteUgcDirectoryRecursive(ctx, path)** — удалить директорию рекурсивно.

Типы: **UgcFile** (UgcUgcFileFull), **UgcFileListItem** (UgcUgcFileListItem).

---

## Права на файлы

- **listUgcFilePermissions(ctx, path)** — список прав на файл.
- **hasUgcFilePermission(ctx, path, permission)** — проверка права.
- **createOrUpdateUgcFilePermission** — выдать/обновить право.
- **deleteUgcFilePermission** — удалить право.

Тип **UgcFilePermission** (UgcUgcFilePermission).
