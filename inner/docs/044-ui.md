# Модуль @app/ui: Блоки и JSX интерфейса

Краткий справочник по модулю `@app/ui` — блоки UI (компоненты JSON-разметки), jsx, Fragment и типы для построения интерфейса. Типизация: `node_modules/@app/ui/index.d.ts`.

## Назначение

**@app/ui** — блоки и типы для описания UI в формате платформы (чатium-json): кнопки, тексты, изображения, контейнеры и т.д. Экспортирует функцию **jsx** для сборки дерева блоков, **Fragment**, реэкспорты из lib/chatium-json, **attachMedia** для вложений, а также множество типов блоков и пропсов. Используется в роутах и компонентах, возвращающих JSON-разметку.

---

## Основные экспорты

- **jsx(block, props?, ...children)** и **jsx** namespace — построение узлов разметки (BlockFactory, JsxNode).
- **attachMedia(params)** — действие прикрепления медиа (AttachMediaAction).
- **Fragment** — реэкспорт из lib/chatium-json/v1.
- Реэкспорты компонентов и типов из lib/chatium-json (v1, v2) — блоки, иконки, layout и др.

Точный список блоков и типов см. в `node_modules/@app/ui/index.d.ts`.
