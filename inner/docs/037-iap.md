Краткий справочник по модулю `@app/iap` — покупки внутри приложения (подписки, продукты). Типизация: `node_modules/@app/iap/index.d.ts`.

## Назначение

**@app/iap** — API для работы с покупками в приложении: проверка наличия продукта у пользователя (hasPurchasedProduct), дата истечения подписки (getIapExpirationDateByUser), поиск покупок по пользователю или сессии и по originalTransactionId. Тип покупки — UgcIapPurchase (экспортируется как IapPurchase).

---

## Основные экспорты

- **hasPurchasedProduct(ctx, userId?, productId?)** — проверка, куплен ли продукт.
- **getIapExpirationDateByUser(ctx, userId?, productId?)** — дата истечения подписки.
- **findAllIapsByUser(ctx, userId?)** — все покупки пользователя.
- **findAllIapsByUserOrSession(ctx, userId?)** — покупки по пользователю или сессии.
- **findIapByOriginalTransactionId(ctx, txnId)** — поиск по ID транзакции.
- **findIapsByOriginalTransactionIds(ctx, txnIds)** — поиск по списку ID.
- **IapPurchase** (UgcIapPurchase) — тип записи о покупке.

Полный список полей и сигнатур см. в `node_modules/@app/iap/index.d.ts`.
