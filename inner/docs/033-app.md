# Модуль @app/app: Вызовы между приложениями

Краткий справочник по модулю `@app/app` — вызов функций и межприложенные вызовы в UGC. Типизация: `node_modules/@app/app/index.d.ts`.

## Назначение

**@app/app** позволяет вызывать функции приложения по пути: в текущем аккаунте или в другом приложении (target). Используется для межсервисных вызовов и вызова джобов/роутов приложения по строковому пути.

---

## Экспорты

- **runAppFunction(ctx, targetApp, path)** / **runAppFunction(ctx, targetApp, path, params)** — выполнить функцию приложения по пути с опциональными параметрами.
- **runInterAppCall(ctx, targetApp, path)** / **runInterAppCall(ctx, targetApp, path, params)** — межприложенный вызов.
- **runAppFunctionInCurrentAccount(ctx, path)** / **runAppFunctionInCurrentAccount(ctx, path, params)** — вызов в текущем аккаунте.
- **runInterAppCallToCurrentAccount(ctx, path)** / **runInterAppCallToCurrentAccount(ctx, path, params)** — межприложенный вызов в текущий аккаунт.
- **InternalCallTarget** — тип цели вызова (строка или `InternalCallTargetCurrentAccount`).
- **InternalCallTargetCurrentAccount**, **InterAppCallCurrentAccount** — константа `'CurrentAccount'`.
