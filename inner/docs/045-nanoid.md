# Модуль @app/nanoid: Генерация уникальных ID

Краткий справочник по модулю `@app/nanoid` — генерация коротких уникальных идентификаторов. Типизация: `node_modules/@app/nanoid/index.d.ts`.

- [Назначение](#назначение)
- [Экспорты](#экспорты)
- [Связанные документы](#связанные-документы)

---

## Назначение

**@app/nanoid** — генерация уникальных строковых ID для использования в приложении (сущности, токены, ключи). Две области: привязанная к аккаунту (accountNanoid) и общая для приложения (nanoid / appNanoid).

---

## Экспорты

- **accountNanoid(ctxOrName)** — сгенерировать ID в области аккаунта. Принимает RichUgcCtx или строку (имя области).
- **nanoid** (реэкспорт appNanoid) — сгенерировать ID в области приложения (глобальный или привязанный к приложению).

Возвращаемое значение — строка, пригодная для использования как уникальный идентификатор.
