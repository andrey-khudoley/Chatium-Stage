# Runbook

## Назначение

Практическая инструкция «как жить с проектом»: деплой, обновления, перезапуск, логи.

## Когда обновлять

- При изменении процесса деплоя/обновления.
- При появлении новых точек мониторинга или журналирования.
- При изменении путей админ‑инструментов.

## Деплой

- **Среда/платформа**: Chatium workspace
- **Артефакты**: исходный код проекта в папке `./p/gc/partnership/`
- **Процесс**:
  1. Код автоматически синхронизируется с сервером при сохранении файлов
  2. Проект доступен по URL: `https://s.chtm.aley.pro/p/gc/partnership/`
  3. Для форсированного обновления используйте команду `git push` (если используется Git)

## Обновление

1. Внести изменения в код.
2. Прогнать smoke‑проверку:
   - `/` — главная страница
   - `/login` — страница входа
   - `/admin` — админ‑панель
   - `/tests` — интерактивные тесты
   - `/tests/ai` — JSON API для автоматических проверок
3. Убедиться что все тесты в `/tests/ai` зелёные (обязательно согласно стандартам).

## Перезапуск при проблемах

- В Chatium перезапуск происходит автоматически при изменении файлов
- Для принудительного перезапуска обновите любой файл проекта
- Если проблема в данных — проверить таблицы `ProjectSettings`/`ProjectLogs` в Heap
- Системные логи доступны через `/s/dev/logs` (требуется авторизация)

## Где смотреть логи

- UI админки: `/admin` → вкладка логов (требуется роль Admin)
- Хранилище: Heap таблица `ProjectLogs` в `./tables/projectLogs.table`
- Системные логи платформы: `https://s.chtm.aley.pro/s/dev/logs` (требуется авторизация)
- Live-обновления логов через WebSocket в админ-панели

## Быстрые проверки после релиза

- ✅ Открывается `/` и подгружаются настройки проекта
- ✅ `/login` показывает форму входа
- ✅ `/admin` доступен под Admin‑ролью, счётчики и логи отображаются
- ✅ `/tests` показывает интерактивный интерфейс тестирования
- ✅ `/tests/ai` возвращает JSON со всеми тестами (статус должен быть `passed`)
- ✅ `/api/admin-settings` (GET/POST) отвечают без ошибок
- ✅ `/api/admin-logs` (POST) возвращает логи с фильтрацией

**⚠️ КРИТИЧЕСКИ ВАЖНО**: Перед релизом ВСЕ тесты в `/tests/ai` должны быть зелёными!
