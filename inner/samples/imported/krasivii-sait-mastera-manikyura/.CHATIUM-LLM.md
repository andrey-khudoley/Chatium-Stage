# Сайт мастера маникюра

**Версия:** 1.0.0  
**Дата создания:** 2025-10-21

## Описание проекта

Красивый современный сайт мастера по маникюру с возможностью онлайн-записи на прием.

## Основные функции

- 🏠 **Главная страница** с информацией о мастере
- 💅 **Услуги** с описанием и ценами
- 📸 **Портфолио** работ мастера
- ⭐ **Отзывы** клиентов
- 📅 **Форма записи** на прием
- 📱 **Адаптивный дизайн** для мобильных устройств

## Структура файлов

```
/index.tsx                 - Главная страница сайта
/pages/MainPage.vue        - Vue компонент главной страницы
/components/Header.vue     - Шапка сайта с навигацией
/components/Hero.vue       - Главный баннер
/components/Services.vue   - Раздел услуг
/components/Portfolio.vue  - Галерея работ
/components/Reviews.vue    - Отзывы клиентов
/components/BookingForm.vue - Форма записи
/components/Footer.vue     - Подвал сайта
/tables/bookings.table     - Таблица записей
/api/bookings.ts           - API для обработки записей
/styles.tsx                - Общие стили сайта
```

## Таблицы

### bookings

Хранит записи клиентов на прием:

- name (string) - Имя клиента
- phone (string) - Телефон клиента
- service (string) - Выбранная услуга
- date (string) - Дата записи
- time (string) - Время записи
- comment (string) - Комментарий клиента
- status (string) - Статус записи (pending, confirmed, completed, cancelled)

## API маршруты

- `POST /api/bookings/create` - Создание новой записи
- `GET /` - Главная страница сайта
