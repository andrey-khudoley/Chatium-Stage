# Ping Pong Game - Project Documentation

## Overview

A casual browser-based Ping Pong game built with Vue.js and HTML5 Canvas. Features single-player mode with AI opponent.

## Architecture

### Files Structure

- `index.tsx` - Main entry point, exports `indexPageRoute`
- `pages/PingPongGame.vue` - Main game component with full game logic

## Game Features

### Core Mechanics

- **Player Control**: Mouse movement (desktop) or touch drag (mobile)
- **AI Opponent**: Adaptive AI with difficulty setting (70% reaction speed)
- **Ball Physics**: Dynamic speed increase on paddle hits, angle-based deflection
- **Scoring**: First to 5 points wins
- **Game States**: Start screen, playing, paused, game over

### Technical Details

**Ball Physics**

- Speed increases by 5% on each paddle hit
- Max speed cap: 8 units
- Angle deflection based on hit position on paddle
- Collision detection for paddles and walls

**AI Logic**

- Tracks ball when moving towards AI side
- Difficulty controlled via reaction speed multiplier
- Bounded within canvas limits

**Canvas Rendering**

- Responsive sizing (max 800x600, adapts to screen)
- 60 FPS game loop using `requestAnimationFrame`
- Dark theme with gradient background

## UI Components

- Score display (player vs AI)
- Start/Pause/Resume buttons
- Game over overlay with winner announcement
- Control hints for mouse/touch

## State Management

Uses Vue 3 Composition API with reactive refs:

- `playerScore` / `aiScore` - Score tracking
- `gameStarted` / `paused` / `gameOver` - Game state flags

## Styling

- Gradient purple/blue theme
- Responsive design with mobile breakpoints
- Neon-style UI with glowing effects
- Touch-friendly controls
