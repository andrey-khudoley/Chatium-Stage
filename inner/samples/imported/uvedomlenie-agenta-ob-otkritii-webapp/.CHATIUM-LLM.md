# WebApp Opened Events Handler

Приложение для отправки событий открытия Telegram WebApp в ИИ-агента.

## Структура проекта

- `index.tsx` - главная страница с интерфейсом настроек
- `pages/SettingsPage.vue` - компонент настроек
- `api/settings.ts` - API для работы с настройками и агентами
- `hook/sender.ts` - обработчик хука @sender/webapp-opened
- `tables/settings.table` - синглтон таблица настроек

## Функциональность

### Настройки:

- Выбор канала сендера (Telegram бота) из списка доступных в аккаунте
- Выбор агента из списка существующих в аккаунте
- Создание нового агента
- Режим: только первое посещение или каждое (isNewUser)
- Шаблон сообщения с переменной [userData]
- wakeAgent - будить ли агента при получении события

### Обработка событий:

- Хук @sender/webapp-opened слушает события с payload: `{userId: string, isNewUser: boolean}`
- Проверяет настройки из синглтон таблицы
- Получает данные пользователя по userId через findUserById
- Ищет Telegram ID пользователя через findIdentities
- Получает/создает chat и person через getOrCreateChat с externalId (Telegram ID) и данными пользователя
- chainKey определяется по приоритету: `person?.uid || user.id || chatId`
- Передает chatId в chainParams для агента
- Отправляет сообщение в выбранного агента через pushMessageToChain
- Параметр wakeAgent берётся из настроек (по умолчанию true)
