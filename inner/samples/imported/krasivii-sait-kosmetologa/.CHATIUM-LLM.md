# Документация проекта: Сайт косметолога

## Версия: 1.1.0

## Описание

Современный адаптивный сайт для косметолога с красивым дизайном и всеми необходимыми секциями.

## Архитектура

### Структура файлов

```
/index.tsx                      # Точка входа приложения
/pages/MainPage.vue            # Главная страница
/components/Header.vue         # Шапка сайта с навигацией
/components/HeroSection.vue    # Главный экран с призывом к действию
/components/ServicesSection.vue # Секция с услугами
/components/AboutSection.vue   # Секция "О косметологе"
/components/GallerySection.vue # Галерея работ
/components/ContactSection.vue # Контакты и форма связи
/components/Footer.vue         # Подвал сайта
/styles.tsx                    # Общие стили сайта
/api/leads.ts                  # API для работы с заявками
/tables/leads.table            # Таблица для хранения заявок
```

## Основные возможности

- Адаптивный дизайн (mobile-first)
- Красивая типографика и цветовая схема
- Плавная прокрутка по секциям
- Секция услуг с описанием и ценами
- Галерея работ
- Форма обратной связи с сохранением заявок в таблицу
- Уведомления администратора о новых заявках
- Интеграция с FontAwesome для иконок

## Технологии

- VueJS 3.5.13
- Tailwind CSS 3.4.16
- FontAwesome 6.7.2
- HTML-JSX для серверного рендеринга

## Цветовая схема

Профессиональная схема с зеленым акцентом:

- Основной: #8FB969 (зеленый)
- Вторичный: #A8C585 (светлый зеленый)
- Акцент: #7AA054 (темный зеленый)
- Темный: #4a4a4a (темно-серый)
- Светлый: #f7f7f7 (светлый фон)

## База данных

### Таблица заявок (leads)

Хранит информацию о заявках от клиентов:

- `name` - Имя клиента
- `phone` - Телефон клиента
- `email` - Email клиента (необязательно)
- `message` - Сообщение от клиента (включает выбранную услугу)
- `status` - Статус заявки (по умолчанию 'new')

## API Endpoints

### POST /api/leads

Создание новой заявки от клиента.

**Параметры запроса:**

```json
{
  "name": "Имя клиента",
  "phone": "+7 (900) 123-45-67",
  "email": "client@example.com",
  "message": "Текст сообщения"
}
```

**Ответ при успехе:**

```json
{
  "success": true,
  "leadId": "xxx"
}
```

**Функционал:**

- Валидация обязательных полей (имя и телефон)
- Сохранение заявки в таблицу
- Отправка уведомления администратору
