# Лендинг-тест для мастеров педикюра

Версия: 1.1.0
Дата создания: 2025-10-13
Последнее обновление: 2025-10-15

## Описание проекта

Интерактивный лендинг в виде теста для мастеров маникюра и педикюра. Пользователи проходят опрос из 5 вопросов и получают подарок — 147-страничное профессиональное руководство "Мастер педикюра: введение в профессию" от EMI Online.

## Целевая аудитория

- Мастера маникюра и педикюра
- Начинающие специалисты
- Опытные мастера, желающие развиваться в направлении педикюра

## Структура проекта

### Файлы

```
/index.tsx                 - Главный роут с HTML-структурой
/pages/QuizPage.vue        - Vue-компонент с квизом и формой GetCourse
/tables/quiz_leads.table   - Таблица для хранения лидов (опционально)
```

### Основные компоненты

#### 1. Главная страница (index.tsx)

- Подключает Tailwind CSS для стилизации
- Использует шрифт Inter от Google Fonts
- Настраивает градиентную цветовую схему (фиолетовый/розовый)
- Включает метрику Clarity

#### 2. Квиз-компонент (QuizPage.vue)

**Функциональность:**

- Прогресс-бар с процентами выполнения
- 5 вопросов с выбором одного варианта ответа
- Плавные переходы между вопросами
- Возможность вернуться к предыдущему вопросу
- Экран с результатом и формой GetCourse для регистрации
- Интеграция с CRM GetCourse для автоматической обработки лидов

**Форма GetCourse:**

- Поля: имя, email, телефон
- Автоматический сбор UTM-меток (source, medium, campaign, term, content)
- Отслеживание источников трафика (location, referrer)
- Интеграция со счетчиком GetCourse
- Кнопка "Скачать подарок" с защитой от повторных нажатий

**Вопросы:**

1. Делаете ли вы педикюр в своей практике?
2. Какие темы эфиров по педикюру вам интересны? (включая "Профилактика врастания ногтя. Установка Титановой нити")
3. Какой у вас опыт работы с педикюром?
4. С какими сложностями вы чаще всего сталкиваетесь?
5. Что для вас важно в обучении педикюру?

## Дизайн

### Цветовая схема

- **Primary:** #8B5CF6 (фиолетовый)
- **Secondary:** #EC4899 (розовый)
- **Accent:** #3B82F6 (синий)

### Особенности дизайна

- Градиентный фон (фиолетовый → сиреневый)
- Белые карточки с rounded углами и тенями
- Анимации при наведении на варианты ответов
- Плавные переходы между вопросами
- Адаптивный дизайн для мобильных устройств
- **Оптимизирована верстка первого экрана** — карточка с первым вопросом видна без скролла для повышения конверсии
- Компактный заголовок и минималистичное описание подарка

## Призыв к действию

**Главный оффер:**
147-страничное руководство "Мастер педикюра: введение в профессию" от EMI Online

**Содержание подарка:**

- ✓ Классификация услуг и современные техники
- ✓ Анатомия стопы и аппаратные технологии
- ✓ Оснащение рабочего места и выбор инструментов

**УТП:** "Всего 5 вопросов — меньше 2 минут"

## Технологии

- **Frontend:** Vue.js 3 (Composition API)
- **Styling:** Tailwind CSS
- **Backend:** Chatium Framework
- **Шрифты:** Google Fonts (Inter)
- **CRM:** GetCourse (интеграция через форму)

## Особенности интеграции

- Форма GetCourse стилизована под общий дизайн лендинга
- Автоматический сбор и передача UTM-меток в CRM
- Отслеживание источников трафика (location, referrer)
- Счетчик GetCourse для аналитики конверсий
