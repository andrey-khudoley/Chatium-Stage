# Telegram Web App Bot Settings Page

Проект для настройки и управления ботом через разные
каналы связи используя sender (Telegram, WhatsApp, Viber, Avito).
Не модифицируй настройки бота и не создавай новых. Используй каналы связи
которые уже подключены в sender. Все настройки сохранятся автоматически.

Вся информация о пользователях хранится в sender. Таблицы для хранения пользователей
создавать не нужно.

Не надо создавать эндпоинты для получения хуков от телеграма, вотсапа, вайбера и т.п.
Так как этим занимается sender. Он уже пождключен и работает. Надо лишь реагировать на события
в api/hooks.

Не надо делать интерфейсов отображения последних пользователей и чатов. Все это уже реализовано в sender и используется.
Тут только функционал ответов пользователям.

Если надо создать бота для конкретной ниши, то твоя задача придумать промт и предложить его пользователю, сказав что сам
бот уже написан. Надо лишь проверить настройки подключенных каналов.

## Логика работы приложения и его развития

Первое, что мы делаем это подписываемся на событие о получении сообщения в одном из каналов. Это делается в hook/sender.ts.
Канал на который надо подписаться лежит в config/channels.json.

После получения сообщения и проверки, что мы на этот канал связи реагируем мы отправляем ответ. Ответ формируется LLM используя метод
`pushMessageToChainFromSender` из библиотеки `@ai-agents/sdk/process`.

## Структура проекта

- `index.tsx` - главная страница приложения
- `web-app.tsx` - страница Telegram Web App
- `pages/SenderBotSettings.vue` - компонент отображения настроек бота
- `pages/WebAppIndex.vue` - компонент отображения Telegram Web App
- `config/index.ts` - конфигурация
- `config/config.json` - конфигурация подключенных каналов
- `hook/sender.ts` - подписка на событие о получении сообщения в одном из каналов

## Функциональность

### Основные возможности:

- Двухэтапная настройка: агент и телеграм канал
- Создание агента с настраиваемым названием и промптом
- Подключение Telegram бота по токену
- Telegram Web App для мобильных устройств

## Дизайн

Использует Tailwind CSS для создания минимального и лаконичного интерфейса с центрированной формой.
