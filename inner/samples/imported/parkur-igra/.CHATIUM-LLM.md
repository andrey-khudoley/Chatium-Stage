# Паркур Игра

## Описание

Интерактивная игра про паркурщиков с возможностью делать трюки: сальто вперёд, сальто назад, большие прыжки.

## Архитектура

### Основные файлы

- `index.tsx` - главный маршрут приложения
- `pages/ParkourGame.vue` - игровой компонент с Canvas

## Игровая механика

### Управление

**Клавиатура:**

- **Стрелка влево/A** - движение влево
- **Стрелка вправо/D** - движение вправо
- **Пробел** - прыжок
- **Стрелка вверх/W** - сальто вперёд (в воздухе)
- **Стрелка вниз/S** - сальто назад (в воздухе)

**Мобильное управление:**

- **← / →** - кнопки движения влево/вправо
- **🤸** - кнопка прыжка (центральная красная)
- **↑ Вперёд** - сальто вперёд (в воздухе)
- **↓ Назад** - сальто назад (в воздухе)

### Физика

- Гравитация
- Скорость движения
- Вращение персонажа при сальто
- Столкновения с платформами

## Особенности

- Canvas рендеринг для плавной анимации
- Система платформ разной высоты
- Визуализация вращения персонажа
- Счётчик выполненных трюков
- Адаптивное управление для мобильных устройств
- Виртуальные кнопки с тактильной обратной связью
