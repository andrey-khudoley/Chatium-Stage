# Тест по определению стиля макияжа

## Описание проекта

Элегантный интерактивный тест для определения персонального стиля макияжа. Создан для аудитории женщин 20-65 лет, подписчиков школы бьюти-профессий EMI Online. Цель — прогрев перед бесплатным мастер-классом по макияжу для себя.

## Цели

1. Прогреть подписную базу школы бьюти-профессий
2. Привлечь участников на бесплатный мастер-класс по макияжу (https://emi-courses.ru/makeup_yourself)
3. Предоставить ценность через PDF-гайд "Косметичка любителя"
4. Повысить вовлечённость аудитории через интерактив

## Структура проекта

```
/cosmoMakeupTest/
├── index.tsx              # Главная страница с подключением стилей
├── pages/

```

## Функциональность

### Этапы теста

1. **Стартовый экран**

   - Приветствие
   - Описание бонусов (PDF + мастер-класс)
   - Информация о времени прохождения
   - Кнопка начала теста

2. **Прохождение теста**

   - 7 вопросов с 4 вариантами ответов
   - Прогресс-бар
   - Возможность вернуться к предыдущему вопросу
   - Визуальная обратная связь при выборе ответа

3. **Страница результата**
   - Определение типажа (Минималистка/Глам-гёрл/Романтичная муза/Экспериментатор)
   - Описание личности
   - Персонализированные советы по макияжу
   - CTA на получение PDF и регистрацию на мастер-класс
   - Кнопка повторного прохождения
   - Шеринг в соцсети

## Типажи результатов

### 1. Минималистка ✨

- **Характер**: Элегантная простота, естественность, эффективность
- **Советы**: BB-кремы, качественный уход, нюдовые помады, лёгкий контуринг
- **Целевая ценность**: Естественность и универсальность

### 2. Глэм-леди 💎

- **Характер**: Профессионализм, безупречность, качество
- **Советы**: Контуринг, смоки-айз, профессиональные кисти, стойкие текстуры
- **Целевая ценность**: Совершенство и профессионализм

### 3. Романтичная натура 🌸

- **Характер**: Женственность, элегантность, утончённость
- **Советы**: Персиковые румяна, мягкие переходы, нюдовые оттенки
- **Целевая ценность**: Изящество и гармония

### 4. Творческая личность 🎨

- **Характер**: Креативность, индивидуальность, смелость
- **Советы**: Цветные подводки, необычные текстуры, смелые сочетания
- **Целевая ценность**: Уникальность и самовыражение

- Элегантные градиенты
