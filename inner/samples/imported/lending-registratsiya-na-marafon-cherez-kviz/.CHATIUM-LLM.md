# Лендинг с тестом для мастеров маникюра и педикюра

## Описание проекта

Интерактивный лендинг с тестом из 5 вопросов для мастеров маникюра и педикюра. После прохождения теста и заполнения контактных данных пользователь получает доступ к бесплатному видео-уроку по работе с травмированными ногтями и скоростному наращиванию. Дополнительно предлагается регистрация на бесплатный практикум с 4 премиальными техниками.

## Функционал

### 1. Главная страница (/)

- **Приветственный экран** с информацией о подарке
- **Коллаж изображений** "до/после" работы с травмированными ногтями:
  - Мобильная версия: карусель с touch-свайпом и GPU-ускорением
  - Десктопная версия: 3 изображения в ряд
  - Автопрокрутка каждые 3 секунды (останавливается при взаимодействии)
- **5 вопросов теста**:
  1. Какую премиальную технику хочешь освоить?
  2. Что сейчас важнее всего?
  3. С какой сложностью сталкиваешься?
  4. Какой результат хочешь получить?
  5. Что останавливает от развития?
- После прохождения всех вопросов - переход на страницу формы

### 2. Страница формы (/contact-form)

- **Форма сбора контактов**: имя, email, телефон
- Получает ответы из sessionStorage
- После отправки - переход на страницу благодарности

### 3. Страница благодарности (/thankyou)

- **Видео-урок** по работе с травмированными ногтями
- **Описание содержания урока**
- **Блок с предложением** регистрации на "Ноябрьскую Практику"
- **4 премиальные техники**:
  - Скоростное наращивание (от 3500₽)
  - Креативное моделирование - когти (от 5000₽)
  - Выкладной френч (от 4000₽)
  - Пилочный маникюр (от 3000₽)
- **Прилипающая кнопка регистрации** внизу экрана
- **Модальное окно с формой** регистрации на практикум:
  - Поля: имя, почта, телефон
  - Таймер обратного отсчета (2 минуты) с срочностью
  - Перенаправление на практикум после регистрации

## Технические детали

### Таблица quiz_leads

Хранит контактные данные и ответы на вопросы теста:

- `name` - имя пользователя
- `email` - электронная почта
- `phone` - телефон
- `answers` - ответы на вопросы теста (JSON)
- `utm_source` - UTM метка источника
- `utm_medium` - UTM метка канала
- `utm_campaign` - UTM метка кампании

### API endpoints

#### POST /api/quiz~submit-quiz

Сохраняет данные из формы, отправляет уведомление администратору и записывает событие.

**Параметры запроса:**

```json
{
  "name": "string",
  "email": "string",
  "phone": "string",
  "answers": {
    "question1": "string",
    "question2": "string",
    ...
  },
  "utm_source": "string",
  "utm_medium": "string",
  "utm_campaign": "string"
}
```

**Ответ:**

```json
{
  "success": true,
  "leadId": "string"
}
```

### Внешние интеграции

1. **Уведомления администратора** через `@user-notifier/sdk`
2. **Аналитика событий** через `@start/sdk` (событие `quiz_completed`)
3. **Ссылка на практикум**: https://pro.emionline.ru/utm-pro-mk

## UTM параметры

Приложение автоматически собирает UTM метки из URL:

- `utm_source`

- `utm_campaign`

Эти параметры сохраняются вместе с контактными данными для аналитики источников трафика.

## Маршруты

- `/` - Главная страница с тестом (indexPageRoute)
- `/contact-form` - Форма контактных данных (contactFormRoute)
- `/thankyou` - Страница благодарности с подарком (thankYouRoute)

## Механизм работы

1. Пользователь отвечает на 5 вопросов теста на главной странице
2. Ответы сохраняются в sessionStorage
3. После последнего вопроса происходит автоматический переход на `/contact-form`
4. На странице формы пользователь вводит контактные данные
5. Форма отправляет данные вместе с ответами из sessionStorage в API
6. После успешной отправки происходит переход на `/thankyou`
7. sessionStorage очищается
