# Приложение для записи голосовых сообщений

## Описание

Веб-приложение для записи голосовых сообщений, их загрузки в файловое хранилище Chatium и получения ссылки для шаринга.

## Архитектура

### Основные компоненты

- **index.tsx** - главный роут приложения
- **pages/VoiceRecorder.vue** - страница с интерфейсом записи
- **api/file.ts** - API для получения публичных ссылок на файлы

### Особенности iOS

- Используется MediaRecorder API с проверкой поддержки
- Для iOS используется формат audio/mp4 (поддерживается нативно)
- Для других браузеров - audio/webm
- Обработка ошибок при недоступности микрофона

### Функционал

1. Запись голосового сообщения через браузерный API
2. Загрузка записи в файловое хранилище Chatium
3. Генерация публичной ссылки на файл
4. Копирование ссылки в буфер обмена
5. История записанных сообщений (локально в браузере)

### API Routes

- `GET /` - главная страница приложения
- `GET /api/file/public-url/:hash` - получение публичной ссылки на файл по хешу
