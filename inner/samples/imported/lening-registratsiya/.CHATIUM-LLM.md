# Тест скорости мастера маникюра - Документация

## Описание проекта

Интерактивный лендинг с тестом для мастеров маникюра. Помогает оценить скорость работы мастера и предлагает регистрацию на марафон по скоростному наращиванию на верхние формы.

## Основные компоненты

### 1. TestLanding.vue

Главная страница с интерактивным тестом, включающая:

**Этапы (steps):**

- `quiz` - Прохождение теста (5 вопросов)
- `results` - Отображение результатов и предложение марафона
- `registration` - Форма регистрации на марафон (обработка через GetCourse)

**Вопросы теста:**

1. Время на классический маникюр с покрытием
2. Скорость обработки 10 пальцев
3. Количество клиентов в день
4. Скорость наращивания ногтей
5. Использование техник ускорения

**Система оценки:**

- 80-100% - "Мастер-профи" (потенциал до 200 000 ₽/мес)
- 60-79% - "Опытный мастер" (потенциал до 150 000 ₽/мес)
- 40-59% - "Развивающийся мастер" (потенциал до 100 000 ₽/мес)
- 0-39% - "Начинающий мастер" (потенциал до 120 000 ₽/мес)

### 2. Интеграция с GetCourse

**Форма отправляется на:** `https://emi-courses.ru/pl/lite/block-public/process-html?id=2190890252`

**Поля формы:**

- `formParams[first_name]` - Имя (обязательно)
- `formParams[phone]` - Телефон (обязательно)
- `formParams[email]` - Email (опционально)

**Скрытые поля UTM-меток:**

- `formParams[dealCustomFields][1297809]` - utm_source
- `formParams[dealCustomFields][1297810]` - utm_medium
- `formParams[dealCustomFields][1297811]` - utm_campaign
- `formParams[dealCustomFields][1297812]` - utm_term
- `formParams[dealCustomFields][1297813]` - utm_content

**Технические поля:**

- `__gc__internal__form__helper` - URL страницы
- `__gc__internal__form__helper_ref` - URL реферера
- `requestTime` - Время запроса
- `requestSimpleSign` - Подпись для безопасности
- `isHtmlWidget` - Флаг HTML-виджета

**Функционал:**

- Нативная отправка формы напрямую в GetCourse (без v-model)
- Автоматический сбор UTM-меток из URL через nextTick + setTimeout (полный рендеринг Vue → задержка 100ms)
- Отправка статистики на сервер GetCourse через gccounterImgContainer
- Защита от двойной отправки формы (GetCourse механизм)
- Автоматическое заполнение технических полей (location, referrer)
- GetCourse управляет перенаправлением после успешной отправки
- Поля формы используют value="" вместо v-model для корректной работы с GetCourse

## Информация о марафоне

**Название:** "Скоростное наращивание"

**Тема:** Наращивание на верхние формы - скоростная техника

**Преимущества:**

- Проверенные техники ускорения работы
- Секреты наращивания на верхние формы
- Увеличение дохода минимум на 40%
- Продолжительность: 5 дней

## Целевая аудитория

Мастера маникюра, которые хотят:

- Увеличить скорость работы
- Принимать больше клиентов
- Повысить доход
- Освоить скоростные техники наращивания

## Ключевые метрики

- **Время прохождения теста:** ~1 минута
- **Количество вопросов:** 5
- **Конверсия в регистрацию:** отслеживается через таблицу registrations

## Технологии

- Vue.js 3.5.13 (Composition API)
- TailwindCSS 3.4.16
- TypeScript
- Chatium Backend Framework

## Уведомления

Регистрации поступают в GetCourse CRM (аккаунт emi-courses.ru).
Все уведомления и последующая работа с лидами происходит в GetCourse.

## Особенности формы регистрации

- **Крестик закрытия** в правом верхнем углу
- **Таймер обратного отсчета** на 5 минут для создания срочности
- **Поля:** Имя (обязательно), Телефон (обязательно), Email (опционально)
- **Минималистичный дизайн** без подписей над полями
- **Интеграция с GetCourse CRM** для автоматической обработки заявок
- **Автоматический сбор UTM-меток** для аналитики рекламных кампаний (с nextTick для полного рендеринга)
- **Защита от двойной отправки** формы
- Закрытие формы возвращает пользователя к результатам теста
