# Сайт стилиста с возможностью записи на прием

## Описание проекта

Современный одностраничный сайт для профессионального стилиста с возможностью онлайн-записи на консультацию.

## Технологии

- **Frontend**: Vue 3, Tailwind CSS
- **Backend**: Chatium Backend Framework
- **Database**: Heap Tables
- **Notifications**: @user-notifier/sdk
- **Analytics**: @start/sdk

## Структура проекта

### Основные файлы

- `index.tsx` - точка входа, главный роут
- `pages/MainPage.vue` - главная страница с формой записи
- `styles.tsx` - глобальные стили и конфигурация Tailwind
- `api/appointments.ts` - API для обработки записей
- `tables/appointments.table` - таблица для хранения записей

## API

### POST `/api/appointments/create`

Создает новую запись на консультацию.

**Параметры:**

```typescript
{
  clientName: string
  phone: string
  email?: string
  service: string
  date: string
  time: string
  comment?: string
}
```

**Ответ:**

```typescript
{
  success: boolean
  appointment?: Appointment
  error?: string
}
```

## База данных

### Таблица `appointments`

- `clientName` - имя клиента
- `phone` - телефон
- `email` - email
- `service` - выбранная услуга
- `date` - дата консультации
- `time` - время консультации
- `comment` - комментарий клиента
- `status` - статус записи (new, confirmed, completed, cancelled)
- `createdAt` - дата создания записи

## Уведомления

При создании новой записи:

1. Отправляется уведомление владельцу аккаунта через @user-notifier/sdk
2. Записывается событие в аналитику через @start/sdk
