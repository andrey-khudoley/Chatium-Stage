# Документация проекта: Сайт мастера по макияжу

**Версия:** 1.0  
**Дата создания:** 2025-10-21  
**Язык:** Русский

## Описание проекта

Профессиональный одностраничный сайт (лендинг) для мастера по макияжу с системой онлайн-записи на прием.

## Основные функции

1. **Главная страница** - представление мастера и услуг
2. **Портфолио** - галерея выполненных работ
3. **Услуги и цены** - три категории услуг с описанием
4. **Форма записи** - онлайн-запись на прием
5. **Уведомления** - автоматическая отправка уведомлений администратору

## Технологический стек

- **Frontend:** Vue.js 3 (Composition API)
- **Styling:** Tailwind CSS 3.4.16
- **Icons:** Font Awesome 6.7.2
- **Fonts:** Playfair Display (заголовки), Inter (текст)
- **Backend:** Chatium Framework
- **Database:** Heap Tables

## База данных

### Таблица: appointments

| Поле        | Тип    | Описание                                                 |
| ----------- | ------ | -------------------------------------------------------- |
| id          | string | Уникальный идентификатор                                 |
| clientName  | string | Имя клиента                                              |
| clientPhone | string | Телефон клиента                                          |
| service     | string | Выбранная услуга                                         |
| date        | string | Дата записи                                              |
| time        | string | Время записи                                             |
| comment     | string | Комментарий клиента                                      |
| status      | string | Статус записи (pending, confirmed, completed, cancelled) |

## API Endpoints

### POST /api/appointments/create

Создание новой записи на прием.

**Request body:**

```typescript
{
  clientName: string
  clientPhone: string
  service: string
  date: string (YYYY-MM-DD)
  time: string (HH:MM)
  comment?: string
}
```

**Response:**

```typescript
{
  success: boolean
  appointment: AppointmentDto
}
```

### GET /api/appointments/list

Получение списка всех записей (для администратора).

**Response:**

```typescript
AppointmentDto[]
```

## Уведомления

При создании новой записи:

- Отправляется уведомление администратору через `@user-notifier/sdk`
- Записывается событие в аналитику через `@start/sdk`
- Тип события: `appointment_created`

## События аналитики

Событие **appointment_created** включает:

- Данные пользователя (имя, телефон)
- Параметры действия (услуга, дата, время)
