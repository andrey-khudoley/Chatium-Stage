# Chatium — Интерактивная презентация для вебинара

## Описание проекта

Интерактивная slide-deck презентация для вебинара о платформе Chatium. 26 слайдов с навигацией стрелками/клавиатурой/свайпом, wow-анимациями появления элементов, демо-зонами с iframe, glow-эффектами и интерактивными hover-эффектами.

## Архитектура

### Точка входа

- `/index.tsx` — маршрут, подключает `Presentation.vue`
- `/styles.tsx` — глобальные стили, шрифты, анимации, CSS-переменные

### Основной компонент

- `/pages/Presentation.vue` — slide deck: переключение слайдов (←→, свайп, клики), прогресс-бар с анимированными точками, 3D transition-анимации

### Слайды (каждый — отдельный файл)

**Вступление (5 слайдов):**

- `/slides/SlideIntro.vue` — анимированное вступление «Мир Chatium»
- `/slides/SlideWhatIs.vue` — что такое Chatium, три составляющие (Инфраструктура, Ботан, Леночка)
- `/slides/SlideScale.vue` — «+1 000 сотрудников. Ноль собеседований.» — масштабирование через AI-агентов
- `/slides/SlidePrinter.vue` — бегущие ленты: сервисы и ИИ-сотрудники
- `/slides/SlideDemo.vue` — интерактивное демо: чат слева → живой iframe результат справа (LMS, агенты, Авито)

**Инфраструктура — вводный (1 слайд):**

- `/slides/SlideInfraIntro.vue` — провокация «Почему не ChatGPT?», обзор 16+ модулей

**Модули — по одному слайду на модуль (16 слайдов):**

_Клиенты и коммуникации:_

- `/slides/SlideModContacts.vue` — @contacts: единая база клиентов (сбор из всех каналов, объединение дублей, теги, фильтрация)
- `/slides/SlideModSender.vue` — @sender: рассылки + чат-боты + агенты (10+ каналов)
- `/slides/SlideModAutomations.vue` — @automations: автоматические воронки

_Деньги и продажи:_

- `/slides/SlideModPay.vue` — @pay: платежи (ЮКасса, CloudPayments, Invoice, МИР, СБП)
- `/slides/SlideModHeapTables.vue` — Heap Tables: база данных с поиском по смыслам
- `/slides/SlideModAuth.vue` — @app/auth: личные кабинеты и роли

_ИИ и контент:_

- `/slides/SlideModAgents.vue` — @ai-agents: цифровые сотрудники (Леночка)
- `/slides/SlideModStart.vue` — @start: AI-генерация контента (GPT-4.1, Claude)
- `/slides/SlideModKinescope.vue` — @kinescope: видео, вебинары, трансляции

_Под капотом:_

- `/slides/SlideModStorage.vue` — @app/storage: файлы и CDN
- `/slides/SlideModSocket.vue` — @app/socket: мгновенные обновления
- `/slides/SlideModJobs.vue` — @app/jobs: отложенные задачи
- `/slides/SlideModRequest.vue` — @app/request: интеграция с любым сервисом

_Аналитика и рост:_

- `/slides/SlideModTraffic.vue` — @traffic: аналитика и воронки
- `/slides/SlideModHtml.vue` — @html: быстрые страницы и SEO
- `/slides/SlideModI18n.vue` — @app/i18n: мультиязычность + финал «полмиллиарда за подписку»

**Основные блоки (7 слайдов):**

- `/slides/SlideRussia.vue` — почему подходит для РФ
- `/slides/SlideTeam.vue` — команда
- `/slides/SlideBotan.vue` — вайбкодинг + iframe для демо
- `/slides/SlideLenochka.vue` — AI-агенты + iframe для демо
- `/slides/SlideReplace.vue` — замена сотрудников, экономика
- `/slides/SlideCompare.vue` — сравнение (метафора города)
- `/slides/SlideOffer.vue` — итоги + CTA

### Каждый модульный слайд содержит:

- Badge с группой и номером модуля (N / 16)
- Крупное название модуля с gradient-текстом
- Бизнес-описание (💼 блок) — понятно для предпринимателей
- Feature-чипы с иконками (hover-эффекты)
- Цитата-пример из жизни
- Уникальная цветовая схема и glow-orbs

## Навигация

- Стрелки ←→, ↑↓, Page Up/Down, Space — переключение слайдов
- Home/End — в начало/конец
- Свайп на мобильных (50px threshold)
- Клик по точкам прогресс-бара
- Кнопки ←→ в панели управления

## Дизайн-система

- **Шрифты:** Syne (display), Space Grotesk (body), JetBrains Mono (code)
- **Тёмная тема:** bg #020208, уникальный accent-цвет на каждый модуль
- **Каждый слайд:** полноэкранный, step-based анимации появления
- **Демо-зоны:** с iframe для живых проектов
