# Event Landing Page - EMI Beauty Master Class

## Project Overview

Premium event landing page for nail masters registration. Features modern dark theme with gold accents, focusing on practical masterclasses and registration conversions.

## Event Details

- **Dates**: November 11, 12, 13 at 18:00 MSK
- **Price**: 1799 RUB (Free early bird registration)
- **Target Audience**: Beginner and experienced nail masters

## Key Features

- 4 practical masterclasses on premium techniques
- 3 live sessions with beauty industry stars
- Prize draw: 9 EMI cosmetic sets + Multi LED lamp
- Named certificate for participants
- Exclusive EMI brand offers

## Architecture

### Tables

- `registrations.table` - Stores event registrations with participant data

### API Routes

- `POST /register` - Handle registration submissions
- `GET /stats` - Get registration statistics

### Pages

- `/` - Main event landing page

### Components

- `EventLanding.vue` - Main landing page content
- `RegistrationPopup.vue` - Modal popup with registration form
- `FloatingButton.vue` - Animated floating CTA button

## Design System

- **Theme**: Dark premium with gold accents
- **Colors**:
  - Primary: Gold (#D4AF37)
  - Background: Dark (#0A0A0A, #1A1A1A)
  - Accent: Deep burgundy (#8B2332)
- **Fonts**: Modern sans-serif (Inter/Montserrat)
- **Style**: Luxury, elegant, conversion-focused
