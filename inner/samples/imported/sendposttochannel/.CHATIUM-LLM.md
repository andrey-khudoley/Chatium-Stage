# SendPostToChannel Tool

Тул для отправки сообщений в Telegram каналы и группы через ботов-менеджеров.

## Структура проекта

- `tables/settings.table` - singleton таблица для хранения настроек
- `index.tsx` - главная страница с настройками
- `pages/SettingsPage.vue` - компонент страницы настроек
- `api/settings.ts` - API для управления настройками
- `api/tool.ts` - основной тул для ИИ агентов

## Функциональность

### Настройки

- Выбор бота-менеджера из списка доступных
- Ввод ID группы/канала с подсказками недавно активных
- HTML редактор для текста сообщения
- Добавление кнопок со ссылками
- Опция отключения превью ссылок
- Тестирование отправки сообщений

### Таблица настроек

Использует singleton методы:

- `.getSingleton(ctx)` - получение настроек
- `.updateSingleton(ctx, data)` - обновление настроек

Поля: botManagerId, botManagerTitle, groupOrChannelId, messageText, buttons, disableLinkPreview

### API эндпоинты

- `GET /managers` - получение списка Telegram менеджеров
- `POST /group-suggestions` - получение предложений групп
- `POST /save-settings` - сохранение настроек
- `GET /get-settings` - загрузка настроек
- `POST /test-tool` - тестирование отправки
- `sendPostToChannelTool` - основной тул для ИИ агентов (принимает только текст)

### Интеграция с ИИ агентами

Тул автоматически регистрируется в системе агентов как `sendPostToChannel`. Принимает только текст сообщения, все настройки (бот-менеджер, канал, кнопки) берутся из конфигурации.
Настройки загружаются из singleton таблицы при каждом вызове тула.

## Зависимости

- `@sender/sdk` для работы с Telegram API
- Vue 3 для фронтенда
- TailwindCSS для стилей
- FontAwesome для иконок
