# Лендинг с квизом для мастеров педикюра

## Описание проекта

Конверсионный лендинг с интерактивным тестом-квизом из 5 вопросов для мастеров педикюра. После прохождения теста пользователь получает доступ к бесплатному профессиональному руководству на 147 страниц.

## Структура файлов

```
/index.tsx - главная точка входа с роутингом (2 роута: "/" и "/thank-you")
/pages/QuizPage.vue - страница с квизом
/pages/ThankYouPage.vue - страница благодарности с бонусом
```

## Основные функции

### 1. Первый экран (одностраничная структура)

- **Компактная плашка-заголовок** в самом верху:
  - "Тест для девушек, которые хотят больше зарабатывать и жить в свободном графике в новой бьюти-профессии"
  - Краткая и лаконичная формулировка
  - Яркий оранжево-красный градиент
  - Нацелен на женскую аудиторию
- **Компактная информация о бонусе** - всегда видна во время прохождения теста
- Бонус: Профессиональное руководство мастера-педикюра (147 страниц)
- Указание стоимости оригинального курса (15 000 ₽)
- **Квиз начинается сразу** без дополнительных кнопок
- Всё помещается на первом экране без скролла
- **Плавная динамическая смена вопросов** без перезагрузки страницы

### 2. Квиз (5 вопросов)

- Прогресс-бар с процентами
- Плавные переходы между вопросами
- Интерактивные кнопки выбора ответа с визуальной индикацией выбора
- **Навигация "Назад" и "Далее"** для удобства прохождения
- Пользователь выбирает ответ и нажимает "Далее" для продолжения

Вопросы:

1. Работа в бьюти-сфере
2. Опыт в педикюре
3. Желаемый доход через год
4. Предпочтения по повышению квалификации
5. Интересующие мастер-классы

### 3. Финальный экран с подарком (отдельная страница `/thank-you`)

- **Автоматический переход** после ответа на последний вопрос
- Поздравление с анимированной иконкой подарка
- **Краткое описание бонуса** (руководство на 147 страниц)
- Кнопка "СКАЧАТЬ ПОДАРОК"
- **Модальная форма GetCourse** при клике на кнопку:
  - Интеграция с CRM GetCourse (https://emi-courses.ru)
  - Имя (обязательное поле)
  - Email (обязательное поле)
  - Телефон (обязательное поле) с автоматической маской
  - Подсказка: "Укажите ваши данные верно — именно туда придет подарок"
  - Автоматическое отслеживание UTM меток (source, medium, campaign, term, content)
  - Отслеживание referrer и location для аналитики
  - Кнопка "Записаться на интенсив"
  - Защита от множественных отправок (6 секунд блокировки)

## Дизайн

### Цветовая схема

- Primary: Orange-500 (#f97316) - оранжевый
- Secondary: Red-500 (#ef4444) - красный
- Accent: Amber-50/Orange-50 - светло-оранжевый фон
- Text: Gray-800 (#1f2937) - темно-серый для читаемости
- **Высокая контрастность текста на фоне** - никаких розовых текстов на розовом фоне

### Особенности UI/UX

- Белая карточка на градиентном фоне
- Компактный дизайн первого экрана
- Плавные анимации и переходы
- Адаптивный дизайн (mobile-first)
- Иконки FontAwesome
- Двусторонняя навигация (Назад/Далее)
- Визуальная индикация выбранного ответа

### Анимации

- Pulse для иконки подарка на финальном экране
- **Slide-fade transitions** между вопросами (плавный сдвиг + fade)
- Cubic-bezier easing для естественного движения
- Hover эффекты на кнопках
- Transform scale на главной кнопке действия
- **Без резких прокруток** - все переходы происходят внутри контейнера

## UX улучшения

- Информация о бонусе видна с самого начала (мотивация к прохождению)
- Возможность вернуться к предыдущему вопросу
- Четкая визуализация выбранного ответа
- Кнопка "Далее" неактивна до выбора ответа
- На последнем вопросе кнопка меняет текст на "Получить подарок"
- **Автоматический переход на отдельную страницу** после завершения теста
- Плавный переход между страницами без потери контекста

## Роутинг

- `/` - главная страница с квизом (indexPageRoute)
- `/thank-you` - страница благодарности с бонусом (thankYouPageRoute)

## Интеграция с GetCourse

- **Форма**: ID 2189337920 (ltForm8532667)
- **CRM**: https://emi-courses.ru
- **Автоматический сбор**:
  - UTM метки (source, medium, campaign, term, content)
  - Referrer и текущий URL

- **Маска телефона**: автоматическая для RU/UA номеров (если доступна библиотека phoneAutoMask)
