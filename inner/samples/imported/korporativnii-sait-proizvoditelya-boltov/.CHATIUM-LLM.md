# Корпоративный сайт БолтПром

Версия: 1.0.0  
Дата создания: 2025-10-21

## Описание проекта

Корпоративный сайт для производителя болтов и крепежных изделий "БолтПром". Одностраничное приложение с современным дизайном, демонстрирующее продукцию компании, преимущества и контактную информацию.

## Технологический стек

- **Backend**: Chatium Framework
- **Frontend**: Vue.js 3.5.13 (Composition API)
- **Стили**: TailwindCSS 3.4.16
- **Иконки**: Font Awesome 6.7.2
- **Шрифты**: Inter, Roboto (Google Fonts)

## Структура проекта

```
/boltManufacturer/
├── index.tsx                 # Главный роут приложения

├── components/

│   ├── HeroSection.vue      # Главный баннер
│   ├── AboutSection.vue     # Раздел "О компании"

│   ├── AdvantagesSection.vue # Раздел с преимуществами

│   └── FooterSection.vue    # Подвал сайта
└── .CHATIUM-LLM.md          # Документация проекта
```

## Дизайн и стилизация

### Цветовая палитра

- **Primary**: #1e40af (синий)
- **Secondary**: #1e3a8a (темно-синий)
- **Accent**: #3b82f6 (голубой)
- **Dark**: #111827 (темно-серый)
- **Light**: #f3f4f6 (светло-серый)

### Ключевые CSS-классы

- `.hero-gradient` - градиентный фон для главного баннера
- `.card-hover` - hover-эффект для карточек продукции
- `.btn-primary` - стиль основной кнопки с градиентом
- `.section-title` - стиль заголовков секций с подчеркиванием
