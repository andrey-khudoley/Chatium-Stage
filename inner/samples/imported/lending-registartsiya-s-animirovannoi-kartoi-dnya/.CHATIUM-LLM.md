# Карта Дня Мастера - Интерактивный Лендинг

## Описание проекта

Интерактивный лендинг для мастеров маникюра с концепцией "Карты Дня", вдохновленной картами Таро, но адаптированной под маникюрную тематику. Основная цель - привлечь аудиторию на практикум "Как стать богатым ТОП-мастером в 2026 году" через необычный и вовлекающий интерактив.

## Концепция

Пользователь заходит на страницу и видит элегантный мистический дизайн. Ему предлагается "вытянуть карту дня", которая даст персональное послание и инсайт для его маникюрного бизнеса. После просмотра карты пользователю предлагается зарегистрироваться на бесплатный 3-дневный практикум с мастер-классами, эфирами со звездами и розыгрышем призов.

## Структура проекта

```
/index.tsx                  - Главный роут с подключением всех библиотек
/pages/CardOfDayPage.vue    - Основная страница с интерактивом
/shared/cards.ts            - Данные карт (7 тематических карт)
/api/registration.ts        - API для регистрации пользователей
```

## Карты

В проекте 7 тематических карт для мастеров маникюра:

1. **Карта Скорости** - о важности оптимизации времени работы
2. **Карта Ценности** - о правильном ценообразовании
3. **Карта Притяжения** - о привлечении идеальных клиентов
4. **Карта Энергии** - о работе без выгорания
5. **Карта Мастерства** - о выходе на новый уровень
6. **Карта Свободы** - об управлении своим временем
7. **Карта Изобилия** - о финансовом изобилии

Каждая карта содержит:

- Название и подзаголовок
- Иконку (эмодзи)
- Уникальное AI-сгенерированное изображение в стиле таро
- Описание/послание
- Инсайт с конкретной пользой
- Уникальный градиент

## Пользовательский путь

1. Пользователь заходит на страницу
2. Видит элегантный дизайн с колодой карт
3. Нажимает "Вытянуть карту дня"
4. Анимация перетасовки (2 секунды)
5. Раскрывается случайная карта с посланием
6. **Через 3 секунды появляется sticky-кнопка с подарком** - широкая кнопка внизу экрана (растянута по всей ширине с отступами), текст центрирован
7. Видит информацию о практикуме с ограничением мест (47 мест)
8. Заполняет форму регистрации GetCourse (виджет встроен через iframe)
9. Получает подтверждение регистрации

## API

### POST /register

Регистрация пользователя на практикум.

**Параметры:**

```typescript
{
  name: string,      // Имя пользователя
  email: string,     // Email
  phone: string,     // Телефон
  cardId: string,    // ID вытянутой карты
  cardTitle: string  // Название карты
}
```

**Ответ:**

```typescript
{
  success: boolean,
  message: string
}
```

## Аналитика

При регистрации отправляется событие `cardRegistration` с информацией о:

- Имени, email и телефоне пользователя
- Какую карту вытянул пользователь
- Времени регистрации

## Преимущества подхода

1. **Необычность** - не стандартная форма регистрации, а интерактивная игра
2. **Вовлечение** - пользователь получает персонализированное послание
3. **Ценность** - каждая карта дает конкретный инсайт и пользу
4. **Естественность** - призыв к регистрации идет после получения ценности
5. **Мистика** - тема карт Таро привлекает внимание и создает интригу
6. **Дополнительная конверсия** - компактная sticky-кнопка с подарком остается видимой при прокрутке и не отвлекает от контента
