# Надежда Фомина — Визитка (лендинг)

## Описание проекта

Конверсионный лендинг-визитка маркетолога-стратега Надежды Фоминой. Одностраничный сайт с мягкой экспертной продажей.

## Архитектура

- Одна страница (`/`) — лендинг из 7 секций
- Нет таблиц, нет API, нет бэкенд-логики
- Чистый фронтенд на Vue-компонентах

## Структура экранов (CJM)

1. **HeroSection** — личный бренд, фото, оффер, CTA
2. **MethodSection** — уникальный подход (5 шагов)
3. **ResultsSection** — кейсы с цифрами (6 кейсов, 4 статистики)
4. **ServicesSection** — форматы сотрудничества (6 услуг)
5. **AboutSection** — личность, фото, сертификаты
6. **TestimonialsSection** — цитаты, принципы, 4 отличия
7. **ContactSection** — soft CTA, ссылки на Telegram

## Визуальная концепция

- Тёмная палитра: warm-950 (#1a1008), charcoal, graphite
- Акценты: amber (#c4915e), gold (#d4a574)
- Шрифты: Cormorant Garamond (display) + Inter (body), font-display: swap
- Стиль: luxury editorial, grain overlay, glass-morphism cards
- Фото Надежды: Hero (деловое, тёмный костюм) и About (мягкое, бежевый пиджак)

## Оптимизации (v2)

- **Прелоадер**: luxury-стиль, инициалы «НФ» с пульсирующей анимацией на warm-950 фоне
- **CLS fix**: font-display:swap, фиксированные line-height на всех заголовках и body-текстах
- **Navbar fix**: класс `.navbar` с CSS transition вместо Tailwind-классов — устранена белая полоса при скролле
- **Scroll-reveal**: IntersectionObserver + классы `.reveal` / `.is-visible` — секции появляются с fade-up при скролле
- **Контрастность**: повышена до WCAG-приемлемого уровня — тексты warm-200/80+, границы карточек amber/18+, divider amber/35
- **Glass-morphism**: усилены границы (amber/18→amber/32 при hover), добавлены inset shadow и box-shadow
- **Hover-эффекты**: `.btn-primary` со sweep-гlow, `.btn-outline` с фоновой подсветкой, `.service-icon-wrap` с glow при hover группы
- **Lazy-loading**: все декоративные/фоновые изображения — `loading="lazy"`, hero-фото — `loading="eager"` + width/height
- **Адаптивность 4K**: media-запросы для 2000px+ и 2560px+ с увеличением базового font-size
- **rel="noopener"**: добавлен на все внешние ссылки (Telegram)
