# Проект template_project

## Что это
Базовый шаблон проекта Chatium с минимальными страницами и документацией.

## Где искать детали
- Техническая информация: `README.md`
- Архитектура: `docs/architecture.md`
- API: `docs/api.md`
- Данные: `docs/data.md`
- Запуск: `docs/run.md`
- Решения: `docs/ADR/`
- История диалогов: `docs/LLM/`

## Тесты (актуализация 2026-03-31)
- `shared/testCatalog.ts` (`// @shared`) — блоки и id тестов для списка и `/api/tests/list`.
- Роуты: `api/tests/list.ts`, `api/tests/unit/index.ts`, `api/tests/integration/index.ts`; старый `api/tests/endpoints-check/` удалён.
- `TestsPage.vue`: юнит / интеграция, метрики, прогон HTTP страниц шаблона, точечный запуск отдельного теста из строки (кнопка play) для unit/integration/http; загрузка UI после события `bootloader-complete` (прелоадер шаблона).

## Критические правила
- Не хардкодить URL, использовать `config/routes.tsx`.
- Документацию обновлять при любых изменениях структуры/логики.
