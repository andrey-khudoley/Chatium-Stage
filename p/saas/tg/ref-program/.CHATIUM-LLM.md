# Проект p/saas/ref (Реферальная программа)

## Что это
Проект реферальной программы на базе шаблона Chatium: главная, админка, профиль, логин; настройки и логи в Heap (таблицы t__saas-ref__*).

## Критические правила
- Не хардкодить URL, использовать `config/routes.tsx`.
- Документацию обновлять при любых изменениях структуры/логики.
- Агрегаты рефералов (заказы/оплаты) хранятся в таблице referral_aggregates; репозиторий referralRepo читает/пишет их только там. Пересчёт по кнопке «Пересчитать агрегаты» в админке: POST /api/admin/recalc-referral-aggregates ставит в очередь джоб с campaignOffset: 0, campaignIndex: 0, referralOffset: 0; джоб (jobs/recalc-referral-aggregates.job.ts) пагинирует кампании по campaignOffset (страницы по 1000), рефералов по referralOffset — обрабатываются все кампании; запись агрегатов по каждому рефералу в джобе сериализована с вебхуками через runWithExclusiveLock. В listReferrals без memory-фильтров агрегаты загружаются только по referralIds текущей страницы (без лимита 5000).
