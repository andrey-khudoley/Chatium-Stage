# Проект p/saas/gateways/gc_api

## Что это
Gateway GetCourse на Chatium: каркас на базе `template_project` с путём `p/saas/gateways/gc_api`. Отдельные Heap-таблицы (`t__gc-api-gateway__setting__8Gp3Qw`, `t__gc-api-gateway__log__6Km9Xp`). Цель — централизованный слой между тонкими клиентами и GC API (`op` + `args`, см. спецификацию Gateway и реестр операций в материалах курса).
