# Проект assistant

## Что это
Проект `p/assistant` в Chatium с минимальными страницами и документацией. Главная с карточками; «Мой журнал» (`./web/journal/`, `JournalPage.vue`, блокнот — `JournalNotebookPane.vue`, API `api/journal/notes/`). «Управление задачами» (`./web/tasks/`, `TasksPage.vue`) — иерархия клиент → проект → задача (боковая панель: приглушённая кнопка «Новый клиент», новый проект — иконка в строке клиента; таблица задач), Heap и API `api/tasks/` (tree, CRUD, reorder), DTO в `lib/tasks-types.ts`. Heap: `Table.update(ctx, { id, ... })`, не три аргумента.
