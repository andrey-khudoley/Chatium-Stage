# Проект assistant

## Что это
Проект `p/assistant` в Chatium с минимальными страницами и документацией. Есть главная с карточками разделов и страница «Мой журнал» (`./web/journal/`, `JournalPage.vue` — боковое меню; вкладка «Блокнот» — `JournalNotebookPane.vue`, заметки в Heap, API `api/journal/notes/` list/create/get/update/delete, UI чтения в панели и модал создания/редактирования).

## Где искать детали
- Техническая информация: `README.md`
- Архитектура: `docs/architecture.md`
- API: `docs/api.md`
- Данные: `docs/data.md`
- Запуск: `docs/run.md`
- Решения: `docs/ADR/`
- История диалогов: `docs/LLM/`

- Не хардкодить URL, использовать `config/routes.tsx`.
- Документацию обновлять при любых изменениях структуры/логики.
