# Проект Ассистент (p/assistant)

## Что это
Проект «Ассистент» на базе шаблона Chatium. Путь: `p/assistant`. Главная с разделами (Календарь, Мой день, Неделя, Привычки, Блокнот), админка, профиль, логин, тесты. Навигация и страницы-заглушки разделов без бизнес-логики.

## Где искать детали
- Техническая информация: `README.md`
- Архитектура: `docs/architecture.md`
- API: `docs/api.md`
- Данные: `docs/data.md`
- Запуск: `docs/run.md`
- Решения: `docs/ADR/`
- История диалогов: `docs/LLM/`

## Критические правила
- Не хардкодить URL, использовать `config/routes.tsx`.
- Документацию обновлять при любых изменениях структуры/логики.
