# Проект Ассистент (p/assistant)

## Что это
Проект «Ассистент» на базе шаблона Chatium. Путь: `p/assistant`. Главная с разделами (Календарь, Мой день, Неделя, Привычки, Блокнот), админка, профиль, логин, тесты. Раздел **Мой день** (только для авторизованных): поля Утро/День/Вечер по дате, главные/дополнительные задачи и бэклог с папками, чекбоксы, DnD; Heap: day_entries, my_day_tasks, backlog_folders; API api/my-day/. Раздел **Блокнот** — список заметок, просмотр/редактирование, markdown-редактор с панелью форматирования и чекбоксами; данные в Heap (notebook_notes), API в api/notebook/.
