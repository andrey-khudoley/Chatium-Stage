# Mentions · Screens

1. **Overview Screen**
- Primary KPI zone with filters and quick actions.
- Dense desktop-first layout with mobile fallback to stacked cards.

2. **Worklist Screen**
- Table/list view with compact/comfortable density.
- Column visibility, saved view selector, bulk actions area.

3. **Detail & Collaboration Screen**
- Split layout: context panel + activity/comments timeline.
- Embedded notes editor (Markdown + WYSIWYG switch).

4. **Configuration Screen**
- Domain-level controls, access blocks, and audit context.
- Token-driven UI options (spacing, color, typography hooks).
