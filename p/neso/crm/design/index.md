# CRM Design · Index

Единая навигация по демо-страницам всех разделов каталога `design/`.

## Как пользоваться
1. Открой нужный раздел через колонку **Demo**.
2. Для детализации переходи в **Screens** и **States**.
3. Возвращайся в индекс через ссылку в начале каждой demo-страницы.
