# BPM workspace showcase states

## Core states

- default: balanced load, no critical SLA breaches
- pressure: multiple red SLA rows, escalation indicators active
- outage: integration automation retries growing, timeline shows failures
- recovery: backlog shrink after automation rollout

## Validation checklist

- dense table mode remains readable
- kanban cards keep legible IDs and SLA badges
- timeline severity colors stay distinguishable in dark and light presets
- builder panels preserve hierarchy with custom preset tokens
