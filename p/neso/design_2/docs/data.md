# Data

This module does not define Heap tables and does not persist data.

## Runtime data sources

- `shared/bpmDemoData.ts` - demo arrays for BPM sections.
- `shared/bpmI18n.ts` - RU/EN text dictionary.
- `shared/themeCatalog.ts` - theme token presets.

All data is static and used only for UI rendering in demo routes.
