# Проект: NeSo BPM UI Kit (`p/neso/design_2`)

## Что это

Чистый BPM-first UI модуль с переиспользуемыми компонентами и токенизированной системой тем.

## Где смотреть

- Основная карточка проекта: `README.md`
- Архитектура: `docs/architecture.md`
- Решение по темам: `docs/ADR/0001-bpm-theme-token-architecture.md`
- Запуск и проверка: `docs/run.md`

## Критические правила

- Не возвращать legacy CRM-компоненты в этот модуль.
- Новые UI-блоки добавлять как отдельные компоненты в `components/bpm/`.
- Тема только через токены (`shared/themeCatalog.ts`), без хардкода цветов в логике компонентов.
- Обновлять `docs/imports.md` при изменении импортов в роутах и страницах.
