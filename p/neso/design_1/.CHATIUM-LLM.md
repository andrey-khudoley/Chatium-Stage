# Проект CRM Design System

## Что это
Дизайн-система для CRM-проекта A/Ley Services. Переработана с нуля: сохранены цветовая палитра и вайб, созданы новые компоненты и визуальные решения.

## Концепция дизайна
**Спецификация:** `DESIGN_SPEC.md`

### 🌙 Тёмная тема «Ночной лес»
- Фон: #05080a, поверхность: #11191b
- Акцент: #afc45f (светлячки)
- Глубина, мягкий контраст, фокус на задаче

### ☀️ Светлая тема «Солнечная листва»
- Фон: #f8f6eb, поверхность: #ffffff
- Акцент: #4f6f2f (листва)
- Воздух, чистота, лёгкая иерархия

## Структура файлов
```
/p/neso/design/
├── index.tsx                    # Главная с выбором темы
├── pages/
│   ├── DesignDemoPage.vue      # Тёмная тема
│   └── DesignDemoLightPage.vue # Светлая тема
├── web/
│   ├── design-demo/index.tsx   # Entry point тёмной темы
│   └── design-demo-light/index.tsx # Entry point светлой темы
├── config/
│   ├── project.tsx             # Настройки проекта
│   └── routes.tsx              # Маршруты
└── shared/
    ├── logger.ts               # Логгер
    ├── logLevel.ts             # Уровни логирования
    └── preloader.ts            # Прелоадер
```

## Критические правила
- Не хардкодить URL, использовать `config/routes.tsx`
- Цвета только через CSS переменные
- Без border (кроме glassmorphism границ)
- Консистентность между темами (одинаковая структура)
