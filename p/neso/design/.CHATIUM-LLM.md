# Проект NeSo Design System

## Что это
Премиальная дизайн-система "Modern Nature Minimal" для личного кабинета NeSo Academy. Вдохновлена природой, солнечным светом и органическими формами.

## Концепция дизайна
**Стиль:** Modern Nature Minimal — сочетание природной эстетики с современными технологиями.

**Ключевые характеристики:**
- Glassmorphism с backdrop-filter blur
- Органические скруглённые формы (24px, 16px, 12px)
- Мягкие многослойные тени
- Природная цветовая палитра (зелень + кремовые тона)
- Плавные анимации (cubic-bezier)
- Отсутствие жёстких рамок

## Темы

### 🌙 Тёмная тема "Ночной лес"
- Глубокие тона: #070b0d → #121a1d
- Биолюминесцентное свечение
- Glassmorphism на тёмном фоне
- Акцент: #92a447 (природная зелень)

### ☀️ Светлая тема "Солнечное утро"
- Кремовые тона: #f5f8f0 → #ffffff
- Солнечные лучи (diagonal gradients)
- Белый glassmorphism
- Акцент: #4a5a24 (насыщенная зелень)

## Структура файлов
```
/p/neso/design/
├── index.tsx                    # Главная с выбором темы
├── pages/
│   ├── DesignDemoPage.vue      # Тёмная тема
│   └── DesignDemoLightPage.vue # Светлая тема
├── web/
│   ├── design-demo/index.tsx   # Entry point тёмной темы
│   └── design-demo-light/index.tsx # Entry point светлой темы
├── config/
│   ├── project.tsx             # Настройки проекта
│   └── routes.tsx              # Маршруты
└── shared/
    ├── logger.ts               # Логгер
    ├── logLevel.ts             # Уровни логирования
    └── preloader.ts            # Прелоадер
```

## Критические правила
- Не хардкодить URL, использовать `config/routes.tsx`
- Цвета только через CSS переменные
- Без border (кроме glassmorphism границ)
- Консистентность между темами (одинаковая структура)
