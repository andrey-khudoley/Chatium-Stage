# Tasks Management

Система управления задачами.

Проект в инфраструктуре Chatium.

## Статус

**В разработке** - проект находится на начальной стадии разработки.

## Структура проекта

```
tasks/
├── index.tsx              # Главная страница (/)
├── package.json           # Зависимости проекта
├── tsconfig.json          # Конфигурация TypeScript
├── vue-shim.d.ts          # Типы для Vue
├── jsx.d.ts               # Типы для JSX
├── .workspace.json        # Конфигурация workspace
├── pages/                 # Vue компоненты страниц
│   └── HomePage.vue       # Компонент главной страницы
├── tests/                 # Unit и интеграционные тесты
│   └── index.tsx          # Страница тестов
└── README.md              # Документация проекта
```

## Дизайн

Проект использует дизайн, идентичный проекту `saas/analytics/telegram/channel`:
- Терминальный стиль с CRT эффектами
- Анимации загрузки (bootloader)
- Тёмная тема с акцентным цветом #d3234b
- Адаптивный дизайн для мобильных устройств

## Технологии

- **Vue 3.5.13** - фреймворк для UI компонентов
- **TypeScript** - типизация
- **TailwindCSS 3.4.16** - стилизация
- **FontAwesome 6.7.2** - иконки
- **Share Tech Mono** - терминальный шрифт
