# Презентация · эфир «Chatium + GetCourse»

## Что это

Интерактивная Vue-презентация (slide deck) на 20 слайдов для часового эфира с Эльвирой про связку **Chatium + GetCourse** после декабря 2025: что случилось, что такое HTTPS-интеграция и как её собрал автор (gateway + тонкий SDK).

Запускается как обычное Chatium-приложение по корневому маршруту `/` (роут — `index.tsx`). Полный текст эфирного скрипта и шпаргалка — рядом, в `elvira-broadcast-slides-spec.md` и `elvira-broadcast-cheatsheet.md`.

## Где что лежит

- **`index.tsx`** — роут страницы, подключает `PresentationHead` (стили) и `Presentation.vue`. Меняется только `<title>`.
- **`styles.tsx`** — глобальные стили: палитра (включая «эфирную» сине-чёрно-cyan-amber тему по спеке), шрифты Manrope/Inter/JetBrains Mono (Syne/Space Grotesk оставлены ради старых компонентов), keyframes, утилиты `.dot-grid`, `.anchor-glow`.
- **`pages/Presentation.vue`** — сам deck: viewport, прогресс-бар с точками, клавиатура/свайп, импорт 20 эфирных слайдов.
- **`pages/PresentationPlan.vue`** — отдельный план старой презентации, в эфирном режиме не используется. Не трогать.
- **`slides/Slide<Name>.vue`** — каждый слайд = отдельный самодостаточный Vue-компонент со scoped-стилями. Эфирные слайды перечислены ниже; остальные `Slide*.vue` — наследие прошлой презентации, оставлены как референс по стилю.

5. `SlideDeclBreaks.vue` — три карточки боли + amber-callout про декабрь 2025.
6. `SlideAnchorHttps.vue` — ⭐ якорь: «Интеграция = разговор по HTTPS».
7. `SlideCableToHttps.vue` — цепочка из 7 шагов от кабеля к HTTPS.
8. `SlideProtocolLayers.vue` — вертикальный стек L2..app, верхняя полоса HTTPS подсвечена.
9. `SlideAnchorHttpsWhy.vue` — ⭐ якорь: «зачем HTTPS» + 2 капсулы (зачем / из чего).
10. `SlideRequestFour.vue` — 2×2 grid: метод / URL / заголовки / тело.
11. `SlideRequestExample.vue` — большой код-блок запроса + аннотации.
12. `SlideResponseStatus.vue` — 5 status-pill'ов + блок про тело ответа.
13. `SlideReadApiSpec.vue` — 4 пункта чтения спеки, акцент на «Примерах».
14. `SlideGcDecember.vue` — таймлайн 3 событий + amber-callout.
15. `SlideGatewayManifesto.vue` — 3 строки манифеста про gateway + декоративная «G».
16. `SlideArchitecture.vue` — ⭐ архитектура: Chatium → SDK → Gateway → GetCourse.
17. `SlideStability.vue` — двухколонник «без gateway» (домино) vs «с gateway» (анкор).
18. `SlideSchoolKey.vue` — ⭐ 3 step-карточки + cyan-callout «Не утечёт в git/промпт/скриншот».
19. `SlideOperationsCatalog.vue` — grid имён операций моно-шрифтом.
20. `SlideAccessForm.vue` — 3 шага доступа + псевдо-QR + бонус-callout.

## Дизайн-система (коротко)

- Палитра по спеке: фон `#0B0F19/#070A12`, surface `#131826`, elevated `#1B2336`, accent-cyan `#00D9FF`, accent-amber `#FFB347`, статусы success/warning/error.
- Шрифты: Manrope (display/H), Inter (body), JetBrains Mono (код, бейджи, операции).
- Сквозной фон — `.dot-grid` (точки 80×80, opacity 0.04).
- ⭐ Anchor-слайды (6, 9, 16, 18) — с `.anchor-glow` и накладным cyan radial-gradient.
- Анимации только две: `fadeInUp` и `fadeInLeft` (240–500мс ease-out). Никаких отскоков.

## Платформенные ограничения

Стандартные ограничения Chatium (см. `chatium-constraints` в .cursor/rules):

- В Vue-слайдах — только клиентский код. Нет импортов из `lib/`, `repos/`, `tables/`. Импорты только из `vue` и других слайдов.
- Слайды `position: absolute; inset: 0` живут внутри `.slide-viewport` в `Presentation.vue`.
- Каждый слайд принимает `defineProps({ active: Boolean })` — оставлено для возможности триггерить анимации по активности.
