# Презентация эфира «Chatium + GetCourse»

Интерактивная Vue-презентация на 20 слайдов, собранная для часового эфира с Эльвирой про связку Chatium и GetCourse в свете декабря 2025: что произошло, как устроена HTTPS-интеграция и какой gateway+SDK автор сделал, чтобы ваш код переживал изменения GC.

## Кому это

- **Спикеру** — открыть в браузере как фон эфира; навигация стрелками, Space, точками прогресс-бара или свайпом на мобильном.
- **Авторам контента/дизайна** — образец того, как из текстовой спеки собирается чисто CSS-deck без картинок.
- **LLM-агенту** — точка входа в проект описана в `.CHATIUM-LLM.md`.

## Структура (одной фразой)

`index.tsx` (роут `/`) → `pages/Presentation.vue` (deck-контейнер) → `slides/Slide<Name>.vue` × 20.
Глобальные стили и палитра — в `styles.tsx`.

## 20 слайдов

В `pages/Presentation.vue` массив `slides` идёт в строгом порядке:

1. Title — «Chatium + GetCourse»
2. Agenda — пять пунктов на час
3. «Мы все сейчас вайб-кодим» (anchor pull-quote)
4. Декларативный vs Императивный
5. Где декларативный ломается (3 карточки + amber-callout)
6. ⭐ «Интеграция = разговор по HTTPS»
7. От кабеля к HTTPS — цепочка из 7 шагов
8. Слои протоколов (вертикальный стек)
9. ⭐ HTTPS — зачем и из чего (главная фраза + 2 капсулы)
10. Запрос — четыре вещи (метод, URL, заголовки, тело)
11. Запрос целиком (большой код-блок + аннотации)
12. Ответ и статус-коды (5 status-pill'ов + тело ответа)
13. Как читать API-спеку (4 пункта, акцент на «Примерах»)
14. Декабрь 2025 у GetCourse (таймлайн)
15. Идея gateway — манифест (3 строки + декоративная «G»)
16. ⭐ Архитектура целиком (Chatium → SDK → Gateway → GetCourse)
17. Стабильность: «без gateway» vs «с gateway»
18. ⭐ Ключ школы — НЕ в коде (3 шага + большой cyan-callout)
19. Каталог операций (grid моно-шрифтом)
20. Форма доступа (3 шага + псевдо-QR + бонус-callout)

Подробное описание каждого — в `docs/03-slides.md`. Полная исходная спека — в `elvira-broadcast-slides-spec.md`, поминутная шпаргалка спикера — в `elvira-broadcast-cheatsheet.md`.

## Технологии

- **Vue 3** в Chatium (SSR + клиентский гидрат). Каждый слайд — `<script setup>` без серверных импортов.
- **CSS-only визуал**: внешних картинок нет. Используются градиенты, дот-грид, glow-эффекты, FontAwesome-иконки и инлайн-разметка.
- **Шрифты Google Fonts**: Manrope (display), Inter (body), JetBrains Mono (код). Из старой версии оставлены Syne и Space Grotesk — для возможного переиспользования старых слайдов.
- **Палитра по спеке**: тёмный сине-чёрный фон, accent-cyan `#00D9FF`, accent-amber `#FFB347`, статусные success/warning/error. Подробно — в `docs/02-design-system.md`.
- **Адаптивность**: на десктопе слайды помещаются в viewport; ниже 1024px и 768px — переход в одну колонку, упрощённые декоративные элементы, скролл слайдов на мобильном.

## Навигация

- ←/→, ↑/↓, PageUp/PageDown, Space — соседние слайды
- Home / End — начало / конец
- Клик по точкам прогресс-бара
- Свайп ≥ 50px на тач-устройствах
