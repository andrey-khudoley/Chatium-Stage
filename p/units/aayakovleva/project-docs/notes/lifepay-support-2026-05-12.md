## Способ 1. Ручное выставление СБП-ссылок в `my.life-pay.ru` (текущий)

Подключено по умолчанию вместе с облачной кассой. Шаги:

первичная настройка ЛК (https://support.life-pay.ru/ru/knowledge_base/article/588/category/5/), включая автоматическое закрытие смены, в разделе «Фискализация → Сотрудники» галочка напротив логина для автоматического формирования чека после оплаты по СБП (важно: автоматический чек поддерживает только одну товарную позицию), в разделе «Фискализация → Настройки» выбрать данные чека для автоматической фискализации (https://support.life-pay.ru/ru/knowledge_base/article/154/category/5/#nastroiki), выставление счёта в разделе «Клиенты → Выставить счёт» (https://support.life-pay.ru/ru/knowledge_base/article/44/category/5/#kakvistavitssalku) формирует динамическую ссылку, есть вариант со статической ссылкой (тогда чек уйдёт на почту, указанную при создании ссылки, и описание в чеке будет всегда одинаковым), регулярные ссылки документированы отдельно (https://support.life-pay.ru/ru/knowledge_base/article/525/category/5/).

Все счета видны в разделе «Продажа». Статусы операции: Проведён (оплачен), Ожидает подтверждения, Не проведён, Отменён.

Ручная фискализация чека: https://lifepay.helpdeskeddy.com/ru/knowledge_base/article/95/category/103/
