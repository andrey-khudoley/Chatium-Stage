# Проект lava_gc_integration (Интеграция Lava.top и GC)

## Что это
Приложение Chatium по пути `p/units/aom/lava_gc_integration`: интеграция Lava.top и GetCourse. Структура унаследована от шаблона (страницы, API настроек и логов, Heap). В `lib/settings.lib.ts` объявлены ключи и геттеры для секретов/URL Lava и GetCourse (`lava_api_key`, `lava_base_url`, `lava_product_id`, `lava_offer_id`, `lava_webhook_secret`, `gc_api_key`, `gc_account_domain`, `gc_service_token`). Мастер на админке: ввод ключа и URL → POST `/api/admin/lava/catalog` (каталог продуктов/офферов) → выбор → сохранение id в настройки; `shared/lavaBaseUrl.ts` и `lib/lava-api.client.ts`.
