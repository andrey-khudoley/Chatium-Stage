# Проект p/units/chatiumclub/client3

## Что это

Сценарий **payment-reaction** (webhook GC об оплате → реакция Леночки через gateway `addCommentToDialog`). Webhook принимается **в самом client3** (ADR-0004 «токен в URL»), не в gateway. Путь: `p/units/chatiumclub/client3`. Heap: `t__chatiumclub-client3__setting__6Nt5Yw`, `t__chatiumclub-client3__log__7Ns0Wy`, `t__chatiumclub-client3__webhook_log__3Ds5Cq`. SDK gateway-клиент скопирован локально (`lib/gateway/`, `shared/`). Спецификация: `docs/specifications/clients/payment-reaction`. Сценарный поток (валидация токена → `normalizePaymentEvent` → `processPaymentEvent` → `invoke('addCommentToDialog')` → запись в WebhookLog) описан в `docs/scenario.md`.
