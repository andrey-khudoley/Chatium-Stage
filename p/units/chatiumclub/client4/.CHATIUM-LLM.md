# Проект p/units/chatiumclub/client4

## Что это

Сценарий **registration-welcome** (webhook GC о регистрации → приветствие через gateway `addCommentToDialog`). Webhook (вариант А) принимается **в самом client4** (ADR-0004 «токен в URL»); polling-вариант (Б) — на gateway, не в этом приложении. Путь: `p/units/chatiumclub/client4`. Heap: `t__chatiumclub-client4__setting__5Ou6Zx`, `t__chatiumclub-client4__log__8Ot1Xz`, `t__chatiumclub-client4__webhook_log__7Vx9Pq`. SDK gateway-клиент скопирован локально (`lib/gateway/`, `shared/`). Спецификация: `docs/specifications/clients/registration-welcome`. Сценарный поток (валидация токена → `normalizeRegistrationEvent` → `processRegistrationEvent` → `invoke('addCommentToDialog')` → запись в WebhookLog) описан в `docs/scenario.md`.
