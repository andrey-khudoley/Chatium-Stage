# Проект p/units/chatiumclub/sdk

## Что это

Тонкий клиент (SDK) Chatium+GetCourse поверх каркаса `template_project`. Путь воркспэйса: `p/units/chatiumclub/sdk`. Heap: `t__chatiumclub-sdk__setting__9Hq2Ws`, `t__chatiumclub-sdk__log__4Jp7Vt`.

Реализованная роль — **тонкая прокладка к gateway-приложению `p/saas/gw/gc`**: SDK по обычному HTTP вызывает публичные эндпоинты `/api/v1/{op}` и `/api/v1/operations` gateway. Спецификация поведения gateway — `p/saas/gw/gc/docs/gateway/gateway-operation-manual.md` (SSOT). Решение «SDK и gateway — разные приложения, общение по HTTP» — manual §12.1.

## Ключевые модули

- `pages/AdminPage.vue` — карточка «Подключение к gateway» для ввода `gateway_url`, `gc_school_host`, `gc_school_api_key` (ключи из `shared/sdkSettingKeys.ts`).
- `lib/gateway/gatewayClient.ts` — `invoke(ctx, { op, args, httpMethod? })` и `getOperationsCatalog(ctx)`.
- `lib/gateway/constants.ts` — префиксы путей и таймаут (`SDK_GATEWAY_REQUEST_TIMEOUT_MS = 15_000`).
- `lib/settings.lib.ts` — три gateway-ключа: `gateway_url`, `gc_school_host`, `gc_school_api_key` (валидация в `setSetting`, аккумулятор `getGatewayClientSettings`).
- `shared/sdkSettingKeys.ts`, `shared/gatewayHttpHeaders.ts`, `shared/v1OpsList.generated.ts` — `// @shared` константы для UI и локальный снимок каталога gateway.
- `api/gateway/invoke.ts`, `api/gateway/operations.ts` — Admin-фасады поверх клиента.
