# .CHATIUM-LLM (lp)

Словарь для LLM по проекту **`p/units/chatiumclub/lp`**.

## Что за проект

Одноэкранный лендинг с формой «Получить доступ к SDK». Запускается на платформе
Chatium как дочерний проект внутри `p/units/chatiumclub/`. Принимает заявки и
сохраняет их в Heap.

## Карта файлов

- `index.tsx` — главная страница (`app.html('/')`), SSR `pages/LandingPage.vue`.
- `pages/LandingPage.vue` — Vue Composition API, форма + UI.
- `api/submit.ts` — `app.post('/').body(...).handle(...)` — приём формы.
- `tables/leads.table.ts` — Heap-таблица `Leads`.
- `config/routes.tsx` — `PROJECT_ROOT`, `ROUTES`, `getFullUrl`, `withProjectRoot`.
- `tsconfig.json`, `jsx.d.ts`, `vue-shim.d.ts`, `.dir.json` — каркас проекта.

## Платформенные правила, которые здесь соблюдены

- `ctx`, `app` — глобальные, не импортируются.
- Логи — `ctx.account.log()`.
- Роут API помечен `// @shared-route` — Vue вызывает его через `submitLeadRoute.run(ctx, ...)`.
- Vue **не** импортирует `lib/` / `repos/` / `tables/` — только маршрут API.
- Heap создаётся и обновляется только на сервере (в `api/submit.ts`).
