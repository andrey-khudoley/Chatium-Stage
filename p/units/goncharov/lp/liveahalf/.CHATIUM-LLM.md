# Жизнь вполсилы — Лендинг вебинара

## Описание
Премиальный лендинг для регистрации на вебинар «Жизнь вполсилы» — онлайн-встреча с врачом и практиком ТКМ Алексеем Волковым. Дизайн глубоко отражает философию традиционной китайской медицины через природные формы, успокаивающую палитру и органичную анимацию.

## Архитектура

### Таблицы
- `tables/registrations.table` — регистрации на вебинар (name, email, phone, UTM-метки)

### API
- `api/register.ts` — POST-эндпоинт регистрации, CRM-интеграция, уведомления, события аналитики

### Страницы
- `index.tsx` — точка входа с SEO-мета-тегами
- `pages/MainPage.vue` — главная страница-сборщик секций

### Компоненты (components/)
- `HeroSection.vue` — первый экран с атмосферным overlay, органическими blob-формами
- `SignsSection.vue` — 7 признаков снижения энергии с природными фоновыми элементами
- `WhySection.vue` — почему привычные усилия не работают
- `ResultsSection.vue` — что получит участник
- `ProgramSection.vue` — программа встречи
- `EnergyLevels.vue` — 3 уровня состояния с плавным цветовым градиентом (серо-коричневый → охра → моховой зелёный)
- `PracticalTips.vue` — режим / питание / движение
- `ImportanceSection.vue` — тёмный блок с усиленным контрастом, пульсирующими свечениями, градиентной подсветкой
- `ApproachSection.vue` — основа подхода ТКМ с органическими формами на заднем плане
- `SpeakerSection.vue` — о спикере Алексее Волкове
- `AudienceSection.vue` — кому подойдёт встреча
- `FormatSection.vue` — формат, цена, CTA
- `FinalCTA.vue` — финальный блок дожима
- `RegistrationModal.vue` — модальное окно с glassmorphism-эффектом, backdrop-filter blur

### Стили
- `styles.tsx` — расширенная палитра (moss, ochre, slate), анимации "breatheIn", органические blob-формы

## Визуальная концепция (обновлена)

### Цветовая палитра
- **Moss** (моховой зелёный): акценты, витальность, активное состояние
- **Ochre** (охра): CTA-кнопки, переходные состояния, привлечение внимания
- **Slate** (тёмный сланец): основной текст, контрастные блоки
- **Sage** (шалфей): вспомогательные элементы, мягкие акценты
- **Cream/Warm**: фоновые градиенты, тепло, уют

### Типографика
- **Cormorant Garamond**: увеличенный letter-spacing (0.015–0.02em), жирность 600–700, премиальность
- **Source Sans 3**: body-текст, чистота, читаемость

### Анимации
- `breatheIn` — появление элементов как "вдох", плавное масштабирование + движение
- `flowInLeft/Right` — боковое появление с микромасштабированием
- `expandIn` — расширение блоков с эффектом глубины
- `float` — медленная левитация blob-форм (20s цикл)
- Cubic-bezier (0.34, 1.56, 0.64, 1) для упругих, естественных переходов

### Графические элементы
- **Organic blobs**: размытые формы с border-radius 40%/60%/70%/30%, blur(60px), opacity 0.08
- **Градиенты**: многослойные (3+ цвета), создают глубину вместо плоских заливок
- **Glassmorphism**: backdrop-filter blur(20px), полупрозрачные белые панели с лёгкой границей
- **Свечения**: пульсирующие blur-эффекты в ImportanceSection для фокусировки внимания

### UI-компоненты
- **CTA-кнопки**: охра-градиент с анимированным background-position, тени с ochre-тонами, hover-эффект масштабирования
- **Модальное окно**: glassmorphism, скруглённые углы 28px, мягкая тень
- **Карточки**: backdrop-blur, 2px границы с низкой opacity, hover-трансформации
- **Шкала энергии** (EnergyLevels): визуальный градиент от slate → ochre → moss с маркерами-индикаторами

## Ключевые решения
- Философия ТКМ отражена через природные формы и цвета
- Глубина через многослойные градиенты вместо плоских цветов
- "Дыхание" в анимациях — элементы появляются как естественный вдох
- Контраст тёмных и светлых секций для визуальных пауз
- Интеграции: CRM, уведомления админу, writeWorkspaceEvent
