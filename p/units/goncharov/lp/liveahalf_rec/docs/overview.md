# Liveahalf Rec — технический обзор

## Назначение
Проект отдаёт одну страницу записи вебинара с сохранением визуального стиля исходного лендинга.

## Текущая структура
- `index.tsx` — серверный вход и HTML-обёртка страницы.
- `pages/MainPage.vue` — UI с текстом о записи вебинара и iframe Kinescope.
- `styles.tsx` — шрифты, Tailwind-конфигурация, цветовая палитра и базовые CSS-классы.

## Роутинг
- `GET /` — единственная страница проекта.

## Ограничения
- Проект не содержит backend-API, форм регистрации, таблиц и админских роутов.
- Встраивание видео зависит от доступности внешнего источника `kinescope.io`.
