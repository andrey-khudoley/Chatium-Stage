# Liveahalf Rec

Минимальная версия лендинга «Жизнь вполсилы» с одной страницей записи вебинара.

## Что делает проект

- Отдаёт единственный роут `GET /`.
- Показывает заголовок «Запись вебинара» и встроенный плеер Kinescope.
- Сохраняет визуальный стиль исходного лендинга (палитра, типографика, фоновые элементы).

## Основные файлы

- `index.tsx` — точка входа приложения и SEO-мета.
- `pages/MainPage.vue` — единственная страница с встраиваемым видео.
- `styles.tsx` — общие стили и Tailwind-конфигурация.
