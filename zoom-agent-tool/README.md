# Zoom Agent Tool

Интеграция Zoom для AI-агентов на Chatium: OAuth Server-to-Server, создание встреч, список облачных записей, админ-страница настроек.

Подробности по файлам и зависимостям — в [.CHATIUM-LLM.md](./.CHATIUM-LLM.md) и в [docs/](./docs/) (при появлении модульной документации).

Проверка типов вместе с корнем workspace: `npx -p typescript@5 --yes tsc --noEmit -p tsconfig.json` из корня репозитория `s.chtm.khudoley.pro`.
