# /commit

Вызови навык **`auto-commit`** через **Skill tool** и передай ему `$ARGUMENTS`.

Навык сделает:

1. Соберёт все незакоммиченные изменения (`git status`, диффы, untracked-файлы).
2. Сгруппирует их в отдельные коммиты по логическим единицам (проект / фича / назначение).
3. Сгенерирует для каждого коммита сообщение на русском в формате
   `type(scope): описание` + список изменений (заголовок одной строкой,
   пустая строка, пункты `- …`).
4. Закоммитит каждую группу явными путями (без `git add -A`).
5. Запушит текущую ветку (если не передан `--no-push`).
6. Вернёт список созданных коммитов и результат пуша.

## Аргументы (`$ARGUMENTS`)

- `--no-push` — только коммиты, без пуша.
- свободный текст — подсказка по группировке или формулировкам (учесть при работе навыка).

## Важно

- Не запускай `/check` автоматически — это решает пользователь до коммита (CLAUDE.md §5).
- Не вмешивайся в работу навыка вручную: вся логика группировки и формулировок — в `auto-commit`.
