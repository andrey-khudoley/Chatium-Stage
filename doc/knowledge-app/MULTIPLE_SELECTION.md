# Множественный выбор файлов - Knowledge App

## Обзор

Добавлен функционал множественного выбора документов с возможностью группового удаления.

## Возможности

### 1. Выбор документов

**Через checkbox:**
- Checkbox в заголовке таблицы - выбрать/снять выбор со всех документов
- Checkbox в каждой строке - выбрать/снять выбор с конкретного документа

**Через Shift:**
- Выберите первый документ
- Зажмите Shift и кликните на последний документ
- Все документы между ними будут выбраны

**Перетаскиванием мыши:**
- Нажмите и удерживайте левую кнопку мыши на первом документе
- Проведите мышью по другим документам (не отпуская кнопку)
- Все документы под курсором будут выбраны

### 2. Панель действий

При выборе одного или нескольких документов появляется панель с действиями:
- **Количество выбранных** - отображает количество выбранных документов
- **Удалить выбранные** - удаляет все выбранные документы с подтверждением
- **Отменить выбор** - снимает выбор со всех документов

### 3. Визуальные индикаторы

- Выбранные строки подсвечиваются синим цветом
- Панель действий появляется с анимацией
- Иконки и счетчики показывают прогресс операций

## Технические детали

### Компоненты

**DocsListPage.vue** - основной компонент с таблицей документов:
- `selectedDocuments` - Set с ключами выбранных документов
- `lastSelectedIndex` - индекс последнего выбранного документа (для Shift)
- `isMouseSelecting` - флаг режима выбора мышью
- `mouseSelectionStart` - начальный индекс при выборе мышью

### Функции

- `toggleSelectAll()` - переключение выбора всех документов
- `toggleDocumentSelection()` - переключение выбора одного документа (с поддержкой Shift)
- `handleRowMouseDown()` - начало выбора мышью
- `handleRowMouseEnter()` - продолжение выбора мышью
- `handleMouseUp()` - завершение выбора мышью
- `clearSelection()` - снятие выбора со всех документов
- `deleteSelected()` - удаление выбранных документов

### Стили

- `.selection-panel` - панель с действиями над выбранными файлами
- `.doc-checkbox` - стили для checkbox
- `.doc-row.selected` - подсветка выбранной строки
- Адаптивные стили для мобильных устройств

## Использование

### Выбор нескольких файлов

1. **Вариант 1 - Галочки:**
   - Установите галочку напротив нужных документов
   - Или используйте галочку в заголовке для выбора всех

2. **Вариант 2 - Shift:**
   - Кликните на первый документ
   - Зажмите Shift и кликните на последний
   - Все документы между ними выбраны

3. **Вариант 3 - Перетаскивание:**
   - Нажмите ЛКМ на первом документе
   - Не отпуская, проведите по остальным
   - Все документы под курсором выбраны

### Удаление выбранных файлов

1. Выберите один или несколько документов любым способом
2. В появившейся панели нажмите "Удалить выбранные"
3. Подтвердите удаление в диалоговом окне
4. Дождитесь завершения операции

## Безопасность

- Все операции требуют роль Admin
- Удаление требует явного подтверждения
- Отображается количество файлов перед удалением
- Показывается прогресс и результат операции

## Адаптивность

- На мобильных устройствах панель действий отображается вертикально
- Checkbox всегда видны на всех разрешениях
- Выбор мышью работает на touch-устройствах
